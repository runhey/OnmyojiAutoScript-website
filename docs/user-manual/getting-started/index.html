相关项目 Relative Repositories​

OAS 基于 Alas 框架开发

  • Alas: 碧蓝航线的自动化脚本
  • SRC: 星铁速溶茶，崩坏：星穹铁道脚本，基于下一代Alas框架。
  • NikkeAutoScript: 胜利女神：NIKKE 自动日常脚本
  • AAS: 蔚蓝档案自动化脚本
  • MAA: 明日方舟小助手，全日常一键长草
  • FGO-py: 全自动免配置跨平台开箱即用的Fate/Grand Order助手.启动脚本,上床睡觉,养肝护发,满加成圣诞了解一下?
  • OAS-website: OAS 的文档网站，使用 docusaurus 构建
  • ppocr-onnx: OCR 库，基于 onnxruntime 和 PaddleOCR
  • gurs: 基于赛贝尔曲线模拟滑动轨迹, 引入其轨迹模拟人手滑动
